# 🚀 Quick Start Guide

Get up and running in 5 minutes!

## Step 1: Install Dependencies (1 minute)

Open terminal in project directory and run:
```bash
npm install
```

## Step 2: Local Storage (0 minutes)

No database setup needed. Data is stored in your browser’s localStorage.

## Step 3: (Optional) Enable AI Chat

Create `.env` and add your Gemini key:
```
GEMINI_API_KEY=your_api_key_here
```

## Step 4: Start Server (30 seconds)

```bash
npm start
```

You should see:
```
Port: 3000
http://localhost:3000
```

## Step 5: Access Application (Immediate!)

Open browser:
```
http://localhost:3000
```

## 🐛 Issues?

### Port 3000 already in use?
```bash
# Use different port
PORT=3001 npm start
```

### Local data not saved?
Make sure your browser is not blocking localStorage and you’re not in private browsing mode.
