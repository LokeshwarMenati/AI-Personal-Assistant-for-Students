# 🚀 START HERE

Welcome to your **AI Personal Assistant for Students** app!  
This version uses **localStorage**, so there’s **no database setup**.

Want AI chat? Add `GEMINI_API_KEY` to `.env`.

## ⏱️ Quick 3‑Step Setup
```bash
1. npm install
2. npm start
3. Open http://localhost:3000
```
