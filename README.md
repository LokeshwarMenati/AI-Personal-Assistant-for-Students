# 🤖 AI Personal Assistant for Students

A modern AI-powered web application that helps students manage their studies efficiently with features like AI chat, notes management, task tracking, and study planning.

---

## 🚀 Features

- 🔐 Authentication (Login & Register)
- 💬 AI Chat Assistant (Gemini-powered)
- 📝 Notes Manager (Create, Edit, Delete)
- ✅ To-Do List (Task tracking with due dates)
- 📅 Study Planner (Schedule sessions)
- 🎙️ Voice Input (Speech-to-text chat)
- 🕒 Live Clock
- 🎨 Responsive UI with Dark Mode

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- AI Integration: Gemini API
- Storage: LocalStorage
- Tools: VS Code, Git, GitHub
